# NerdMiner TUI

> Minerador Bitcoin solo em Go com interface TUI (Bubbletea).

## Instalação

### go install (requer Go 1.22+)
```bash
go install github.com/nerdminertui/nerdtui/cmd/tui@latest
```

### Download direto do binário (Exemplo para Linux AMD64)
```bash
curl -fsSL https://github.com/nerdminertui/nerdtui/releases/latest/download/nerdtui-linux-amd64.tar.gz | tar xz
sudo mv nerdtui /usr/local/bin/
```

Você pode encontrar os binários para outras plataformas (macOS, Windows, ARM) na página de [Releases](https://github.com/nerdminertui/nerdtui/releases/latest).

### Docker
```bash
docker pull ghcr.io/nerdminertui/nerdtui:latest
```
